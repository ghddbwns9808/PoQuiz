# 🙌 포켓몬 퀴즈 안드로이드 앱 PoQuiz

## ❓ 프로젝트 소개
- PoQuiz는 포켓몬 사진을 보고 이름을 맞추는 포켓몬 퀴즈 안드로이드 앱입니다.  
- 포켓몬 도감 번호에 따라 총 4개의 난이도로 분류됩니다.  
- 난이도별 랭킹 시스템을 제공합니다.
